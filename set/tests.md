# Testing with RVM

RVM allows you to run tests across multiple ruby versions:

```
~/projects/ruby/coulda [git:master] $ rvm 2.0.0,2.1.1 do rake test
```

Visit [sets do](/set/do/) for more information on invoking `rvm do`.
