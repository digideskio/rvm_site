# Uninstalling Rubies

To remove an RVM installed ruby do:

```
$ rvm uninstall 1.8.6
Removing /Users/wayne/.rvm/rubies/ruby-1.8.6-p399...
```

Uninstalling a ruby removes only the install and leaves the source.
